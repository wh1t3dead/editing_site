Админка без сервера

Редактор контента для GitHub Pages.

Эта страница не публикует изменения сама. Она помогает быстро собрать новые темы и видео, а затем скачать обновлённый content.js для коммита в GitHub.

Важно: GitHub Pages статический. Полноценная админка с сохранением прямо на сайте потребует отдельную авторизацию и внешний бэкенд. Здесь сделан бесплатный и простой вариант: редактируешь, скачиваешь файл, заменяешь его в репозитории.

Какие ссылки поддерживаются

youtube - обычная ссылка на YouTube, сайт сам превратит её во встроенное видео.

vimeo - ссылка на Vimeo.

mp4 - прямой путь к видеофайлу, например если положишь ролик в репозиторий.

link - просто кнопка на внешний кейс.

placeholder - карточка без ссылки, если ролик ещё не загружен.
